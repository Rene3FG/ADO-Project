# Sistema de Control de Autobuses (SCA)

Sistema web multiusuario para el control operativo de autobuses
en el área de mantenimiento de ADO Oaxaca.

Desarrollado como parte del proyecto Practicum I —
Universidad Anáhuac Oaxaca, 2026.

---

## Descripción

El SCA digitaliza y centraliza el seguimiento de autobuses
dentro de las distintas áreas de trabajo de la planta de
mantenimiento ADO. Permite registrar unidades, controlar su
movimiento entre áreas, visualizar disponibilidad en tiempo
real y mantener sincronización bidireccional con el archivo
Excel existente.
